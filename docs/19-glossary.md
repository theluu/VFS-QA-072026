# Glossary

- Candidate: mau duoc pipeline chon de con nguoi review.
- Candidate mining: qua trinh tim/cat candidate tu video.
- Manifest: JSON liet ke samples va metadata.
- Annotation: nhan do reviewer tao.
- Ground truth: nhan da review va duoc chap nhan.
- Event label: loai su kien.
- Event boundary: thoi diem bat dau/ket thuc event.
- Proxy clip: clip ngan phuc vu review.
- Background clip: clip khong giao voi candidate event.
- Source video: video goc.
- Sample: mot don vi review trong manifest.
- `sample_id`: khoa duy nhat cua sample.
- Reviewer: nguoi gan nhan.
- Coverage: muc do bao phu dataset.
- Dataset card: tai lieu mo ta dataset release.
- Schema version: version contract JSON.
- Artifact: file dau ra co the ban giao.
- Release: goi artifact da validate.
