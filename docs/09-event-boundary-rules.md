# Event Boundary Rules

- Event bat dau tai frame dau tien co bang chung hanh vi.
- Event ket thuc tai frame cuoi cung con bang chung hanh vi.
- Neu event bi che khuat, dung boundary tot nhat va comment ro.
- Neu doi tuong ra khoi khung hinh, ket thuc tai thoi diem roi khung hinh tru khi event tiep tuc ro rang.
- Nhieu nguoi trong mot clip can comment.
- Nhieu event trong cung clip dung label chinh va comment event phu.
- Event dai hon clip dung `needs_review`.
- Camera doi goc dung `camera_movement` neu ROI bi lech thuc te.
- Sai so timestamp POC chap nhan trong khoang 1000 ms.
- Escalate khi label, boundary hoac context khong ro.
