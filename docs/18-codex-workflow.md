# Codex Workflow

## One Task at a Time

Khong yeu cau Codex cung luc dung repo, thiet ke schema, viet candidate mining, viet annotation UI, viet test va release neu khong chap nhan tradeoff ve do sau.

## Before Coding

Codex phai xac dinh:

- File se sua.
- Requirement dang xu ly.
- Dependency.
- Acceptance criteria.
- Test se chay.
- Assumption phat hien duoc.

## After Coding

Codex phai bao:

- File da tao.
- File da sua.
- Command da chay.
- Test pass/fail.
- Output tao ra.
- Blocker con lai.
- Task tiep theo.
