# Annotation Tool Spec

## Required Features

- Chon manifest.
- Validate manifest khi load.
- Hien thi clip.
- Hien thi sample metadata.
- Previous/next sample.
- Chon event label.
- Nhap event start/end.
- Chon ground-truth status.
- Nhap reviewer.
- Nhap comment.
- Save progress.
- Resume progress.
- Export annotation.
- Validate truoc export.

## Not Hard-coded

- Label list.
- Status list.
- Dataset path.
- Output path.
- Reviewer name.

Gia tri phai doc tu config hoac manifest.
